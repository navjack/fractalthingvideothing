     9) 	What is this?
    10) 	~~~~~~~~~~~~~
    11) 	olc::PixelGameEngine is a single file, cross platform graphics and userinput
    12) 	framework used for games, visualisations, algorithm exploration and learning.
    13) 	It was developed by YouTuber "javidx9" as an assistive tool for many of his
    14) 	videos. The goal of this project is to provide high speed graphics with
    15) 	minimal project setup complexity, to encourage new programmers, younger people,
    16) 	and anyone else that wants to make fun things.
    17) 
    18) 	However, olc::PixelGameEngine is not a toy! It is a powerful and fast utility
    19) 	capable of delivering high resolution, high speed, high quality applications
    20) 	which behave the same way regardless of the operating system or platform.
    21) 
    22) 	This file provides the core utility set of the olc::PixelGameEngine, including
    23) 	window creation, keyboard/mouse input, main game thread, timing, pixel drawing
    24) 	routines, image/sprite loading and drawing routines, and a bunch of utility
    25) 	types to make rapid development of games/visualisations possible.
